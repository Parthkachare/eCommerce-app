# E-Commerce Frontend Application

## Project Overview

This project is a fully responsive E-Commerce Frontend Application developed using React. The application provides product listing, product details, shopping cart management, checkout functionality, and simulated user authentication.

---

## Technologies Used

- React
- React Router DOM
- Context API
- FakeStoreAPI
- Local Storage
- CSS

---

## Features

- Product Catalog
- Product Details
- Shopping Cart
- Quantity Management
- Checkout Process
- Login and Logout
- Local Storage Persistence
- Responsive Design
- Error Handling
- Lazy Loading
